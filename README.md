# PPLX Stats

Расширение для браузера, которое собирает и анализирует статистику ваших чатов в Perplexity.ai.

## Возможности

- Сканирование всех чатов
- Статистика по количеству сообщений и токенов
- Данные хранятся локально в браузере

## Установка

```bash
git clone https://github.com/Krimex1/pplx-stats.git
```

Или скачайте ZIP через кнопку **Code** → **Download ZIP**

### Загрузка в браузер

1. Откройте `chrome://extensions/` (Chrome) или `edge://extensions/` (Edge)
2. Включите **Режим разработчика**
3. Нажмите **Загрузить распакованное расширение**
4. Выберите папку с файлами (где находится `manifest.json`)

## Использование

1. Откройте [perplexity.ai](https://www.perplexity.ai) и войдите в аккаунт
2. Перейдите в библиотеку чатов и прогрузите все чаты
3. Кликните на иконку расширения в панели инструментов
4. Нажмите **Запустить анализ**

Расширение автоматически:
- Соберёт все ссылки на беседы
- Проанализирует содержимое
- Покажет статистику по моделям и токенам

**Разрешения:**
- `storage` — сохранение статистики
- `activeTab`, `scripting` — работа со страницами
- `https://www.perplexity.ai/*` — доступ только к Perplexity

## Ограничения

- В чатах считаются только прогруженные заранее сообщения
- Подсчёт токенов приблизительный
- Анализ больших библиотек может занять время
